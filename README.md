# Selectra React Redux Boilerplate 

This application is a Boilerplate that should be used to create new React project<br/>

# Project Stack

The entire project stack is kept updated to its latest versions.

# Instalation

```
npm install

npm run start
```

# Test:

```
npm run test
```

# e2e:

```
npm run e2e
```

# Eslint:

```
npm run linter
```
